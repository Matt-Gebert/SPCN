Welcome to my own solution for our Church website.
Content has been split into various directories for navigational ease.


/css		-		Styling Scripts
/fonts		-		Glyphicons & UI Elements for JQuery
/images		-		Glyphicons & Icons for JQuery UI
/js			-		Javascript Libraries
/src 		-		Website Resources & Content
	/img 	-		Content Images
	/msg 	-		Imported Text Bodies
/less		-		Files for Font-Awesome's CSS Libraries
/scss		-		Similar purpose to 'less'.
